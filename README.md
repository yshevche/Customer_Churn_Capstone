# Customer Churn Data Analysis 

## Description

Customer churn is important because it can boost profits, improve brand identity and encourage growth. When you can identify your customers pain points you can tweak the products to be the solution they're looking for. Creating relationships with your customers is essential to having a low customer churn rate, when customers are emotionally invested theyâ€™re more likely to choose you over competitors.

This dataset comes from kaggle and customer churn is defined as a marketing metric that describes the number of customers that leave a business over a specific period of time. Each user is assigned a prediction value that estimates their state of churn based on user demographic information, browsing behavior, and historical purchase data among other features.
